# Security

This repository is a static asset collection. It does not include a backend service or production credentials.

## Reporting

If you find a security issue in scripts, workflow configuration, or packaged files, open a GitHub issue with a minimal reproduction. Do not include secrets in issue text, screenshots, or uploaded files.

## Scope

- Build scripts under `scripts/`
- GitHub Actions workflow files
- Packaged JSON metadata
- Static demo HTML

Out of scope:

- Local ignored `resources/` source material
- Third-party GitHub or Codex platform behavior
