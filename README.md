# Light Pets

简体中文 | [English](README.en.md)

Light Pets 是一套可以直接安装到 Codex 的桌面宠物资源。它们把灯泡、灯带、开关、传感器、面板、网关、场景控制等智能照明元素变成小宠物：有静态预览图，也有状态动图，方便你先看效果再挑选安装。

当前仓库包含 **94 只 PET**，分成 **4 组**：精选单宠、Yeelight 高精修新系列、Yeelight 家居灯光系列、Yeelight Pro 系列。

## 快速预览

下载或克隆仓库后直接打开 demo：

```bash
git clone https://github.com/Yeelight/light-pets.git
cd light-pets
open "demo/index.html"
```

如果浏览器限制本地文件访问，可以启动一个静态预览服务：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080/demo/`。demo 里可以搜索、按系列筛选，并查看每只 PET 的 9 个状态动图。

## 安装一只 PET

每只 PET 的安装文件都在 `packages/<pet-id>/` 下。通常复制这两个文件到 Codex 的自定义宠物目录即可：

- `pet.json`
- `spritesheet.webp`

旁边的 `qa/<pet-id>/` 目录放的是完整接触表和 9 个状态动图，适合安装前先看清楚效果。

## 怎么选

- 想先体验完整度高、风格均衡的，选 **精选单宠**。
- 想看这次新做的高精修、多风格、有产品特色的，选 **Yeelight 高精修新系列**。
- 想要更偏家庭灯光氛围的，选 **Yeelight 家居灯光系列**。
- 想按 Pro 产品类型挑，比如开关、射灯、灯带、传感器、网关，选 **Yeelight Pro 系列**。
