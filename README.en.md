# Light Pets

[简体中文](README.md) | English

Light Pets is a ready-to-use Codex desktop pet collection inspired by smart lighting. Bulbs, lightstrips, switches, sensors, panels, gateways, scene controls, and comfort-lighting ideas are reimagined as small animated companions. Each pet includes a static preview and animated state GIFs so you can inspect it before installing.

This repository currently includes **94 pets** across **4 collections**: featured standalone pets, the new Yeelight Refined Hatch Pets, a home-lighting batch, and a Yeelight Pro batch.

## Quick Preview

Clone the repository and open the demo:

```bash
git clone https://github.com/Yeelight/light-pets.git
cd light-pets
open "demo/index.html"
```

If your browser blocks local file access, start a static server:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080/demo/`. The demo supports search, collection filters, contact sheets, and 9-state GIF previews.

## Install A Pet

Each installable pet lives under `packages/<pet-id>/`. Copy these two files into your Codex custom pet directory:

- `pet.json`
- `spritesheet.webp`

Use the matching `qa/<pet-id>/` folder to inspect the full contact sheet and animated state GIFs before installing.

## How To Choose

- Start with **Featured Single Pets** for a small polished set.
- Pick **Yeelight Refined Hatch Pets** for the newest high-polish, multi-style characters with stronger product personality.
- Pick **Yeelight Hatch Pet Full Batch** for residential lighting moods.
- Pick **Yeelight Pro Hatch Pet Batch** when you want product-type coverage across switches, spotlights, lightstrips, sensors, gateways, and panels.
