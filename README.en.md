# Light Pets

English | [简体中文](README.md)

Light Pets is a ready-to-use Codex desktop pet collection inspired by smart lighting. Bulbs, lightstrips, switches, sensors, panels, gateways, and other lighting objects are reimagined as small animated companions. Each pet includes a static preview and animated state GIFs so you can inspect it before installing.

This repository currently includes **82 pets** across 3 groups: featured single pets, Yeelight residential lighting pets, and Yeelight Pro pets.

## Preview Quickly

Clone or download the repository, then open the local gallery:

```bash
git clone https://github.com/Yeelight/light-pets.git
cd light-pets
open "demo/index.html"
```

If your browser blocks local file access, start a static server instead:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080/demo/`. The demo lets you search, filter by collection, and inspect all 9 animated states for each pet.

## Install One Pet

Each installable pet lives in `packages/<pet-id>/`. In most cases you only need to copy these two files into your Codex custom pet directory:

- `pet.json`
- `spritesheet.webp`

For example, `yeelight-cozy-glow` uses:

```text
packages/yeelight-cozy-glow/pet.json
packages/yeelight-cozy-glow/spritesheet.webp
```

The matching `qa/<pet-id>/` folder contains preview sheets, state GIFs, and extra preview material so you can inspect the pet before using it.

## How To Choose

- Start with **Featured single pets** if you want the most polished standalone options first.
- Choose **Yeelight residential lighting pets** for home-lighting moods such as pendants, lightstrips, bedside lamps, and desk lamps.
- Choose **Yeelight Pro pets** when you want pets grouped around Pro product types such as switches, spotlights, lightstrips, sensors, gateways, and panels.

Each row below includes a static preview and an idle animation. Open the demo or click “all GIFs” to see every state.
